Дневник погоды

Нуруллин Амир Рамилевич

Краткое описание:
GUI-приложение "Weather Diary" позволяет вести дневник погодных наблюдений. Вы можете:
- добавлять записи с датой, температурой, описанием и признаком осадков;
- просматривать все записи в удобном списке;
- фильтровать записи по дате (точное совпадение) и по температуре (больше заданной);
- сохранять записи в JSON-файл и загружать их обратно.

Программа проверяет корректность вводимых данных (формат даты ДД.ММ.ГГГГ, температура – число, описание не может быть пустым).

Примеры использования:
- Добавление записи:
Дата: 15.04.2025
Температура: 12.5
Описание: Солнечно
Осадки: галочка не установлена
Нажмите «Добавить запись». Запись отобразится в списке.

- Фильтрация:
В поле «Дата» введите `15.04.2025` и нажмите «Применить фильтр» — покажутся записи только за этот день.
В поле «Температура >» введите `10` — останутся записи с температурой выше 10°C.
Фильтры можно комбинировать.

- Сохранение/загрузка:
Меню «Файл» → «Сохранить как JSON» сохранит текущий список в выбранный файл.
«Загрузить из JSON» — загрузит ранее сохранённые записи.

Требования:
Не требует установки дополнительных библиотек, используется только стандартная библиотека Python (tkinter, json).

Как запустить приложение:
1. Убедитесь, что у вас установлен Python 3.6 или новее.
2. Скачайте репозиторий и поместите папку из архива в одну директорию, которую затем нужно открыть в IDE и запустить файл main.py.
